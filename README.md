1. What is JSX, and why is it used?
JSX hocche JavaScript er vitore HTML-er moto syntax use korar ekta system.
Eta React component ke easily UI design korte help kore.
Code readable o maintain kora onek easy hoy.

2. What is the difference between State and Props?
Props parent component theke child component e data pass korar jonno use hoy.
State hocche component er nijer data ja change hote pare.
Props read-only kintu state change kora jay.

3. What is the useState hook, and how does it work?
useState React er ekta hook ja functional component e state manage korte use hoy.
Eta ekta state variable ebong ekta function return kore state update korar jonno.
State change hole component automatically re-render hoy.

4. How can you share state between components in React?
State share korte parent component e state rakhte hoy.
Tarpor props diye child component gulo te pass kora hoy.
Eta ke lifting state up bola hoy.

5. How is event handling done in React?
React e event handling JSX er moddhe event handler use kore kora hoy.
Jemon onClick, onChange, onSubmit etc.
Event er sathe ekta function call kora hoy action handle korar jonno.
